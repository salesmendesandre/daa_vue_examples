# Node.js y Express: primeros pasos

Ejemplos del capítulo [Introducción a Node.js y Express](https://salesmendesandre.github.io/daa_vue/main/vue/p6c15_node_express.html) del libro **Desarrollo de Aplicaciones Avanzadas con Vue**.

## Carpetas
- `01_first_server_http`: servidor HTTP con el módulo nativo de Node y respuesta de texto plano.
- `02_express`: primer “hola mundo” con Express; inicia con `npm install && node server-express.js`.
- `03_arch_express`: estructura base con `src/app.js`, rutas y middleware de logger; arranca con `npm install && npm run dev`.

Todos los ejemplos escuchan en `http://localhost:4000` (puedes cambiar el puerto en el código si ya está ocupado).
